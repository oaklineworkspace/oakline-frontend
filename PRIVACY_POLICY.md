## 4. Data Security

### 4.1 Security Measures

We implement industry-standard security practices:

**Encryption**:
- TLS 1.3+ for all data in transit
- AES-256 encryption for data at rest
- HTTPS-only connections
- Encrypted password storage (bcryptjs hashing)

**Access Control**:
- Role-based access control
- Row-level security in database
- API authentication and authorization
- Secure session management

**Monitoring**:
- Real-time threat detection
- Regular security audits
- Penetration testing
- Intrusion detection systems

### 4.2 Limitations

While we implement comprehensive security, no system is completely secure. We cannot guarantee absolute security, and you transmit information at your own risk.

You can disable location access through your device settings:

**iOS**:
- Settings → Privacy → Location Services → OaklineBank → Never

**Android**:
- Settings → Apps → OaklineBank → Permissions → Location → Don't allow
